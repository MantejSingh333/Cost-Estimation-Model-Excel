# Cost Estimation Model (Excel)

## Overview
Developed an Excel-based cost estimation model for an AI-powered online writing tool, forecasting monthly operational costs based on student usage, session frequency, and activity complexity across Year 6–8 levels in New Zealand.

## Key Assumptions
- 120,000 students across Years 6–8  
- 80% active user rate  
- 10 sessions per student per month  
- Usage complexity increases by year level  

## Cost Components
- AI input token costs  
- AI output token costs  
- NLP syntax analysis (character-based pricing)  

## Key Result
- Estimated total monthly cost: **$345.60**

## Skills
Cost modeling, forecasting, usage-based pricing analysis, Excel financial modeling
